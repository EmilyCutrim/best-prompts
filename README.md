## PROMPT LIBRARY — Claude AI 🤖✨

#### 📖 Description:

A personal repository to centralize ready-to-use prompts for **Claude** (Anthropic).

Each folder contains a structured prompt for a specific purpose — from personal finance assistants to day-to-day automations. The goal is for anyone to copy a prompt, fill in their own data, and have a working agent in minutes — no coding required.

All prompts follow **prompt engineering** principles: clear role definitions, data blocks separated from logic, and explicit behavior rules for the model.

---

#### 📁 Repository structure:

```
claude-prompts/
├── finance-dashboard/
│   ├── personal-finance-dashboard-prompt.md   ← Main prompt
│   └── finance-dashboard-demo.html            ← Example generated output
├── [other projects]/
│   └── ...
└── README.md
```

Each folder contains:
- A `.md` file with the full prompt and a `[USER_DATA]` block for personalization
- A generated output example (when applicable)

---

#### 🚀 How to use a prompt from this repository with Claude Pro:

##### 1. Access Claude

Go to [claude.ai](https://claude.ai) and sign in. The **Pro plan** is recommended — it provides access to Claude Sonnet and Opus with a larger context window, which is essential for prompts that generate HTML files, extended code, or long analyses.

> ℹ️ The Free plan also works for most prompts, but may hit message limits faster during longer sessions.

##### 2. Start a new conversation

On the Claude home screen, click **"New chat"**.

##### 3. Paste the prompt

Open the `.md` file for the prompt you want from this repository. Copy the **entire content** of the file — from top to bottom, including the `[USER_DATA]` block.

Paste it directly into the Claude message box.

##### 4. Fill in the `[USER_DATA]` block

Before sending, find the data block at the end of the prompt:

```
[USER_DATA]

currency_symbol: $
income:
  - label: Primary salary | amount: 0   ← replace with your value
...

[/USER_DATA]
```

Replace the example values with your real data. You **do not need to change anything outside this block** — all logic and instructions for the model are already defined.

##### 5. Send and wait for the output

Click send. Claude will process the prompt and return the described output — which may be an HTML file, a document, an analysis, or another format specified in the prompt.

##### 6. Use the result

- If the output is an **HTML file**: copy the generated code, save it as a `.html` file on your computer, and open it in your browser.
- If it is a **document or analysis**: read it directly in the conversation or ask Claude to export it in a specific format.
- For **adjustments**: just ask in the same conversation — Claude keeps the context and updates only what's needed.

---

#### 💡 Tips for Claude Pro users:

- **Projects**: use Claude's "Projects" feature to keep your prompts saved and reusable without pasting them again each session.
- **System prompt**: inside a project, you can set the prompt as a *system prompt* — it will activate automatically in every conversation within that project.
- **Iteration**: after the initial generation, you can request changes in plain language. E.g.: *"add a fixed expense of $200 for gym"* — no need to manually edit any code.
